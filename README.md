### Overview

A single page image sharing web application that allows users to upload images/pictures <br/>
and add comments.
Technologies used:
*   HTML/CSS
*   Javacript
*   Node/Express
*   PostgreSQL
*   VueJS
*   Amazon s3
*   Annyang

### Features

-Upload Pictures
-Add Comments
-Virtual Assistant
